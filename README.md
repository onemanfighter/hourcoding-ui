## Installation

```bash
npm install @hourcoding/hourcoding-ui
or
npm i @hourcoding/hourcoding-ui
```

## Usage

Basic component usage syntax:

```jsx
import React from 'react';
import { Button } from '@hourcoding/ui';

const App = () => <Button>Click me</Button>;
```
